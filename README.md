# Poll App Setup

## Backend Setup

Mongodb connection string is defined in .env file 
```sh
cd poll-app
cd backend
yarn install
yarn start
```


## Frontend Setup

```sh
cd poll-app
cd frontend
yarn install
yarn start
```
URLs are defined in URL.js. My backend is hosted on https://poll-app-jbec.onrender.com . Change it to localhost to connect to backend server.
